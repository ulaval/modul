## Mixin `m-font-size('<dimension>')`
| Valeur | Description|
| -- | -- |
| 'h1' | Get the font-size of a level 1 title |
| 'h2' | Get the font-size of a level 2 title |
| 'h3' | Get the font-size of a level 3 title |
| 'h4' | Get the font-size of a level 4 title |
| 'h5' | Get the font-size of a level 5 title |
| 'h6' | Get the font-size of a level 6 title |
| 'base' | Get the base font-size at the root of the site |
| 'xl' | Get the extra large font-size (22px) |
| 'ls' | Get the large font-size (20px) |
| 'md' | Get the medium font-size (18px) |
| 'sm' | Get the small font-size (14px) |
| 'xs' | Get the extra small font-size (12px) |
